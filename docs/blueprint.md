# **App Name**: Mac-Out Recommerce

## Core Features:

- Product Display: Showcase refurbished Apple products with images, titles, descriptions, and a 'Reconditioned & Verified' tag.
- Contact Form: Allow users to send messages via a contact form with fields for name, email, and message.
- Dashboard Authentication: Authenticate access to the dashboard using a secret passcode.
- Equipo Management: Allow employees to add Apple devices to the home page carousel
- Dashboard summarization: Summarize the information in the Dashboard. Use the LLM to evaluate all numbers and values available for 'Equipos en venta', 'Ingresos', 'Gastos', 'Balance neto', and create a summary of 20 to 30 words for the employee. The app should use a tool to retrieve any monetary amounts on the management screens.

## Style Guidelines:

- Primary color: Bright yellow (#FFD700) to convey energy and optimism.
- Background color: Off-white (#FAFAFA) for a clean, modern feel.
- Accent color: Vibrant orange (#FFA500) for calls to action and highlights.
- Headline font: 'Poppins', a modern sans-serif for titles.
- Body font: 'Roboto', a clear and readable font for the text.
- Subtle fade-in animations on scroll to reveal sections.
- Employ a grid layout for product listings.
